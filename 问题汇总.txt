push困难解决方法：
git config --global http.proxy
git config --global --unset http.proxy
git config --global https.proxy
git config --global --unset https.proxy

设置远程仓库地址：
git remote set-url <远程仓库名> http://.../xxx.git
添加远程仓库地址
git remote add origin http://.../xxx.git
